# 电商AI客服助手（淘宝/闲鱼/飞猪版）

一个基于 Chrome Extension Manifest V3 的客服提效插件，默认本地规则处理，不自动发送消息，只提供建议话术。

## 已实现能力（MVP）

- 淘宝/闲鱼/飞猪客服页面消息抓取（仅抓“用户最新一句”）
- 场景识别（尺码/发货/物流/退换/差评安抚/通用，规则分类器V2）
- 话术生成（专业/亲和/强硬三种语气）
- 违禁词检测与高亮（内置词库）
- 一键复制 + 操作日志
- 模型增强模式（可开关，超时自动回退规则模板）
- 误判修正开关（可手动指定场景）

## 项目结构

- `manifest.json`：插件配置与权限
- `content.js`：淘宝/闲鱼/飞猪页面消息监听与提取
- `background.js`：规则引擎、话术生成、日志与存储
- `sidepanel.html` / `sidepanel.css` / `sidepanel.js`：侧边栏 UI 与交互
- `options.html` / `options.css` / `options.js`：独立设置页（高级配置）

## 本地加载

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录下的 `plugin`
5. 打开客服工作页面（淘宝/闲鱼/飞猪），点击插件图标打开侧边栏

## 使用说明

1. 保持客服页面开启，插件会自动监听最新用户消息
2. 在侧边栏选择语气：专业 / 亲和 / 强硬
3. 可开启“收到新消息后自动生成建议”“生成后自动复制到剪贴板”
4. 点击“生成并复制”（主按钮）或“仅生成”，也可等待自动生成
5. 检查违禁词高亮提示
6. 复制后人工粘贴发送（不自动发送）
7. 如需模型配置或误判纠偏，点击“打开高级设置”

快捷键：
- `Enter`：仅生成
- `Ctrl/Cmd + Enter`：生成并复制

## 最新 UI 优化

- 侧边栏升级为“流程卡片化”布局，视觉层次更清晰
- 新增运行状态徽标（待命/处理中/需关注），减少操作不确定感
- 新增场景识别置信度展示，便于判断识别可靠性
- 新增回复生成信息栏（模型耗时/回退原因），排查更高效
- 高级设置页改为双列信息架构，模型配置与纠偏配置更易读

## 无账号演示

如果暂时没有淘宝/闲鱼/飞猪客服账号，可在侧边栏使用“本地测试模式（无店铺也可演示）”：

1. 填写一条测试买家消息（或点示例按钮）
2. 点击“写入测试消息”或“写入并生成”
3. 即可完整体验识别、生成、违禁词检测、复制流程

## 模型增强（可选）

- 默认关闭
- 开启后建议配置：
  - 模型接口地址
  - 接口访问令牌（与服务端 `SERVICE_ACCESS_TOKEN` 一致）
  - 模型超时（毫秒）
- 服务端默认要求令牌鉴权，令牌不正确会回退到本地规则模板
- 接口建议返回 JSON，优先读取如下字段：
  - `reply`
  - `data.reply`
  - `output`
  - `content`
  - `text`
  - `result`
  - `message`
  - `choices[0].message.content`
- 支持设置页“测试模型”快速校验
- 超时或异常会自动回退到本地模板，并显示失败原因

## 安全边界

- 无需店铺账号密码与 API 授权
- 不自动发送消息，仅提供建议文本
- 默认本地规则处理；仅在用户手动开启时调用外部模型接口
